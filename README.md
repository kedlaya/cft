Hints for processing PreTeXt:

- To convert XML to LaTeX:
```
make latex
```
- To convert the resulting LaTeX to PDF:
```
make pdf
```
- To prepare images:
```
make images
```
- To convert XML to HTML:
```
make html
```

You should ignore the `make install` option, as this is for my personal use.
